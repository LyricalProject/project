ikomek - это сервис для поиска лекарств и медицинских услуг по Казахстану. Мы не продаём препараты и не предлагаем услуги, а помогаем быстро найти аптеку или место, где они есть и по комфортной цене.

1. Найдите кнопку «Лекарства» в верхней части сайта и выберите ваш город
2. Начните вводить название препарата в строку поиска и выберите нужный препарат из выпадающего списка
3. Далее сервис покажет аптеки, где имеется искомый препарат. Из аптек выберите комфортную для себя как по местоположению так и по цене
4. По окончанию поиска желательно позвонить в аптеку и уточнить наличие препарата.

Мы работаем над тем, чтобы доставка лекарств i-komek была во всех городах Казахстана. Доставку можно оформить по этой ссылке https://ikomek2020.kz
